Description

DocFetcher est une logiciel libre de recherche de bureau : il permet de rechercher des informations à l'intérieur des documents enregistrés sur votre ordinateur. Autrement dit, c'est un moteur de recherche comme Google, mais pour vos documents personnels. Le logiciel est actuellement disponible pour Windows et Linux.

Comment DocFetcher fonctionne ?

  • DocFetcher crée des index à partir des fichiers. La recherche s'effectue ensuite en une fraction de seconde à partir de ces index. Vous pouvez soit créer des index permanents pour les dossiers comprenant un grand nombre de fichiers rarement modifiés ou des index temporaires pour les petits dossiers qui contiennent des fichiers qui changent souvent.
  • Les index temporaires peuvent être créés en cliquant droit sur un dossier et en choisissant "Rechercher avec DocFetcher". Ils sont automatiquement supprimés lorsque le programme est fermé. Vous pouvez bien sûr demander à DocFetcher de conserver ces index.
  • Le processus d'indexation peut prendre quelques minutes pour les dossiers les plus larges, mais les petits dossiers sont traités en quelques secondes. Sur un ordinateur moyen, 200 documents peuvent être indexés par minute.
  • Une fois que l'index est créé, vous pouvez taper des mots clés dans le champ de recherche de DocFetcher, par exemple "analyse fourier", puis taper sur la touche Entrée. DocFetcher affiche alors tous les documents dans les dossiers sélectionnés qui contiennent les mots recherchés, le plus souvent en moins d'une seconde.

Ce n'est pas tout...

  • Mise à jour automatique des index : Les index sont mis à jours automatiquement lorsque les fichiers sont modifiés, même lorsque DocFetcher n'est pas en fonctionnement. Cela est réalisé grâce à un démon (service) qui surveille les modifications en arrière-plan. Ce démon utilise le CPU de manière minimale : il ne réindexe pas les fichiers modifiés, mais signale simplement à DocFetcher les fichiers qui devront être réindexés la prochaine fois que le logiciel sera utilisé.
  • Une version portable : La version portable fonctionne à la fois sur Windows et Linux. Vous pouvez placer les documents à indexer dans un dossier et ensuite déplacer librement le dossier complet (c'est-à-dire DocFetcher + index + documents) vers un autre ordinateur, un volume crypté (TrueCrypt ou FreeOTFE ), un lecteur de CD-Rom ou un lecteur USB. La version portable peut même être utilisée pour partager un dossier de documents indexés sur un réseau local ou entre plusieurs systèmes d'exploitation sur un ordinateur multi-boot.
  • Détection des paires HTML, telles que "foo.htm" et un dossier nommé "foo_files". Chaque paire sera traitée comme un document unique. Cette fonctionnalité est apparemment inutile, mais elle améliore largement la qualité des résultats lorsque la recherche porte sur des fichiers HTML, puisque les sous-fichiers de la page disparaissent des résultats affichés.
  • Recherche dans le code source des fichiers : Les extensions de fichiers reconnues par DocFetcher peuvent être personnalisées. Vous pouvez donc utiliser le logiciel pour toutes sortes de fichiers texte.

Autres fonctionnalités notables

  • Exclusion de fichiers de l'indexation sur la base d'expressions rationnelles.
  • Multiples opérations réalisables à partir de l'interface de DocFetcher (création de dossiers, déplacements de fichiers...).
  • Panneau de prévisualisation avec surlignage des mots cherchés et navigateur web intégré.
  • Résultats de recherche pouvant être triés et filtrés selon différents critères (types de fichier, taille, chemin, etc.).
  • Raccourci global pour afficher DocFetcher sur le bureau.

Formats de documents supportés

  • HTML et texte brut (personnalisables)
  • Portable Document Format (pdf)
  • Microsoft Office (doc, xls, ppt)
  • Microsoft Office 2007 (docx, xlsx, pptx)
  • OpenOffice.org Writer, Calc, Draw et Impress (odt, ods, odg, odp)
  • Rich Text Format (rtf)
  • AbiWord (abw, abw.gz, zabw)
  • Microsoft Compiled HTML Help (chm)
  • Microsoft Visio (vsd)
  • Scalable Vector Graphics (svg)

Comparaison avec d'autres logiciels de recherche de bureau

Deux éléments distinguent DocFetcher des autres applications de recherche de bureau tels que Google Desktop. Premièrement, il n'indexe que ce que vous voulez et non pas l'ensemble des disques durs. Deuxièmement, il indexe seulement les documents. Les images, la musique, la vidéo, etc. ne sont pas traités. Plusieurs raisons expliquent ce choix :
  • Plus le nombre de fichiers à traiter est élevé, plus le "bruit" dans les résultats de recherche est important. De ce fait, vous constaterez que DocFetcher donne souvent de meilleurs résultats.
  • Moins de fichiers à indexer = indexation plus rapide et charge inférieure du CPU (n'avez-vous jamais été gêné par des programmes qui surchargent l'ordinateur en arrière-plan ?).
  • La puissance de Google et des autres moteurs de recherche Web réside dans l'indexation des documents, parce c'est ce qui permet d'avoir une bonne approximation du contenu d'un document. L'indexation des images ou des vidéos n'est pas forcément utile, car leur contenu réel n'est pas accessible par le biais d'une extraction du texte.
  • Par ailleurs, pourquoi installer encore un autre programme pour accéder aux images, à la musique, aux vidéos, aux courriels, etc. alors que votre logiciel de gestion d'images, votre lecteur multimédia, votre programme de courrier, etc. ont été optimisés depuis de nombreuses années pour ne faire que cela ? Pourquoi utiliser une interface de recherche générique dans ce cas ? C'est pourquoi DocFetcher n'indexe pas les images, ni la musique, ni les vidéos... Que reste-t-il ? Les documents. C'est ce sur quoi DocFetcher est spécialisé. (De toute façon, Launchy est recommandé pour lancer rapidement les applications du menu général.)
Bref, si vous ne vous intéressez pas outre mesure aux interfaces clinquantes, si vous voulez juste un outil libre, ouvert et puissant qui fait uniquement ce qu'il est censé faire, sans violer votre intimité et sans installer des programmes "extras", alors DocFetcher est fait pour vous !
