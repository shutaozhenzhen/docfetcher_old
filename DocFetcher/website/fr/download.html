Logiciel requis : Java Runtime Environment (JRE)

Un environnement Java version 1.6.0 ou plus est nécessaire pour utiliser DocFetcher. Pour connaître la version de JRE de votre système, taper "java -version" dans une invite de commandes.

Vous pouvez télécharger Java pour Windows ici. Si vous utilisez Linux, vérifiez dans le dépôt officiel de votre distribution.

Téléchargements des binaires de la version actuelle de DocFetcher

Remarque : Si vous utilisez un système d'exploitation 64 bits, vous devez remplacer l'environnement Java 64 bits par son équivalent en 32 bits, parce que la version 64 bits n'est pas encore supportée.

Pour un historique des changements depuis les versions précédentes, consultez le changelog.

Plateforme
 Notes de version
docfetcher_1.0.3_win32_setup.exe La version pour Windows fonctionne sur Windows XP et sur les versions ultérieures de Windows. Windows 98 n'est pas supporté. Les autres versions de Windows peuvent fonctionner, mais elles n'ont pas été testées et ne sont pas supportées officiellement.
docfetcher_1.0.3_all.deb La version de Linux de DocFetcher a été testé avec succès avec les environnements GNOME, Xfce et KDE.
Si votre système ne supporte les packages Debian, vous pouvez utiliser la version portable à la place.
docfetcher_1.0.3_portable.zip La version portable de DocFetcher fonctionne sur tous les systèmes d'exploitation Windows ou Linux supportés. Aucune installation n'est nécessaire : il suffit d'extraire le contenu de l'archive dans le dossier de votre choix.
